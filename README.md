### Project Setup

- Install **python 3.6.3**

- Install **pip**

- Install **Django 1.11.7** 
```Bash
$ pip install Django==1.11.7
```

- Install **pillow**

````Bash
$ pip install Pillow
````

- Migrate models
````Bash
$ manage.py migrate [appname]

$ manage.py makemigrations [appname]
````
